# Projet Proba Python

## Requirements

Python 3 with Tkinter

```bash
$ sudo apt install python3.7
$ sudo apt install python3-tk
```

## Installation

Dans le dossier `projet_proba_python`, faire :

```bash
$ python3 -m venv venv
$ source venv/bin/activate
$ python3 setup.py install
```

## Usage

Dans le dossier `projet_proba_python`, faire :

```bash
$ proba
```
